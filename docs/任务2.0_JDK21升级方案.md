### 2.0.2 更新maven-compiler-plugin配置

#### 配置模板

在父pom.xml的pluginManagement中标准化配置：

```xml
<pluginManagement>
    <plugins>
        <plugin>
            <groupId>org.apache.maven.plugins</groupId>
            <artifactId>maven-compiler-plugin</artifactId>
            <version>3.11.0</version>  <!-- 最新支持JDK 21的版本 -->
            <configuration>
                <source>21</source>
                <target>21</target>
                <encoding>UTF-8</encoding>
                <verbose>false</verbose>
                <!-- JDK 21特性启用 -->
                <compilerArgs>
                    <arg>-parameters</arg>  <!-- 支持反射获取参数名 -->
                </compilerArgs>
            </configuration>
        </plugin>
    </plugins>
</pluginManagement>
```

#### 检查清单

- [ ] 确保maven.compiler.plugin版本≥3.11.0
- [ ] source和target都设置为21
- [ ] 编码格式设置为UTF-8

### 2.0.4 JDK 21新特性兼容性检查

#### 检查清单

1. **虚拟线程（Virtual Threads）**
   - 如果使用了线程池，检查是否可以优化为虚拟线程
   - 搜索：`new Thread(`, `Executors.newFixed`, `ThreadPoolExecutor`

2. **文本块（Text Blocks）**
   - 检查是否有多行字符串可以改用文本块
   - 搜索：多行SQL、多行JSON

3. **记录类（Records）**
   - DTO类是否可以改为Records
   - 搜索：`@Data` lombok类

4. **模式匹配（Pattern Matching）**
   - 检查instanceof是否可以改用模式匹配
   - 搜索：`instanceof`

5. **接口新增API**
   - 检查是否使用了过时API
   - 搜索编译警告：`[WARNING] deprecated`

#### 建议优化

**虚拟线程示例**：
```java
// ❌ 传统线程池
ExecutorService executor = Executors.newFixedThreadPool(10);

// ✅ JDK 21虚拟线程
ExecutorService executor = Executors.newVirtualThreadPerTaskExecutor();
```

**记录类示例**：
```java
// ❌ 原始DTO（使用Lombok）
@Data
@NoArgsConstructor
@AllArgsConstructor
public class UserDTO {
    private Long id;
    private String name;
    private String email;
}

// ✅ 使用Records（JDK 21）
public record UserDTO(Long id, String name, String email) {}
```

### 2.0.5 解决编译错误

#### 常见错误和解决方案

**错误1：Cannot access class 'java.lang.System' from module 'java.base'**

原因：模块化配置问题

解决：
```xml
<maven.compiler.release>21</maven.compiler.release>
```

**错误2：使用了JDK 21不存在的方法**

解决：检查Java API文档，使用JDK 21中的替代方法

**错误3：日志框架不支持**

原因：SLF4J或Logback版本过低

解决：升级到最新版本
