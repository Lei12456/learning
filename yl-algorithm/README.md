# yl-algorithm

## 📖 模块介绍
算法实现示例和学习资料，包含常见数据结构和算法的Java实现。

## 🎯 学习内容
- 排序算法（快排、归并、堆排等）
- 查找算法（二分查找、BFS、DFS等）
- 数据结构（链表、栈、队列、树、图等）
- 动态规划、贪心算法等

## 📦 技术栈
- Java 21
- JUnit 5（单元测试）

## 🔗 依赖关系
- 无额外的internal模块依赖

## 💻 项目结构
```
src/
├── main/java/com/yl/algorithm/
│   ├── sort/        (排序算法)
│   ├── search/      (查找算法)
│   ├── datastructure/ (数据结构)
│   └── dp/          (动态规划)
└── test/
```

## 🚀 使用方式
1. 查看具体的算法实现类
2. 阅读代码注释和示例
3. 运行对应的单元测试

## 📚 学习建议
- 理解算法的核心思想，不要死记硬背
- 通过单元测试理解预期行为
- 尝试自己实现变种或优化版本
